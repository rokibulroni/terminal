## 🎯 Overview

Terminal Command Hub is an elite web application built for developers, security researchers, and system administrators to quickly search, bookmark, and explore terminal commands. With built-in features like dynamic global IP substitutions, elegant theming, and an intuitive UI, it transforms tedious reference browsing into an interactive experience.

## ✨ Features

- **Extensive Multi-Vector Database:** Pre-configured with thousands of exhaustively detailed commands spanning strictly across 18 unique operational vectors:
  * *Cloud Infrastructure (AWS, Azure, GCP, Docker/K8s)*
  * *Reverse Engineering & Static Analysis (Ghidra, R2, GDB)*
  * *Mobile Security (Frida, Objection, ADB)*
  * *Hardware & IoT (GPIO, I2C, SPI)*
  * *OSINT, Forensics, Traffic Analysis, Networking, and Linux Administration.*
- **Dynamic IP Substitution:** Never manually replace IPs again! Set a Global IP in the interface, and it instantly overrides placeholder IPs across all commands seamlessly.
- **Lightning Command Palette:** A blazingly fast `Cmd+K` / `Ctrl+K` searchable command palette for rapid access.
- **Modern UI & Theming:** Hand-crafted using Radix UI, Shadcn, and TailwindCSS, boasting flawless Dark/Light modes along with customizable colorful accents (Harvest, Crimson, Forest).
- **Persistent Favorites:** Find an important command? Bookmark it instantly! Favorites stay synced locally.
- **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop viewports.

<a id="quick-start"></a>
## 🚀 Quick Start

Ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

```bash
# 1. Clone the repository
git clone https://github.com/rokibulroni/terminal.git

# 2. Enter the directory
cd terminal

# 3. Install packages
npm install
# or yarn install / pnpm install

# 4. Start the development server
npm run dev
```
Open up [http://localhost:5173](http://localhost:5173) in your browser!
